Fonction Carré

Multiplie un nombre par lui-même

f(x) = x²
🏠 L'image pour comprendre

Imagine que tu as un carré de côté x. Son aire vaut x × x = x². Plus le côté est grand, plus l'aire grandit très vite. C'est pour ça que la courbe monte de plus en plus vite !

📈 La courbe de x² — clique pour voir un point !

👆 Clique sur la courbe pour voir l'image d'un x

📐 Domaine de définition

ℝ (tous les réels)
x peut être n'importe quel nombre

🪞 Parité

PAIRE
f(−x) = f(x)
Symétrie / axe Oy

📉 Minimum

0 en x = 0
x² ≥ 0 toujours !

🎯 Forme de la courbe

Parabole ∪
Sommet en O, ouverte vers le haut

📊 Tableau de variations

x −∞ ⬅ Décroissant ⬅ 0 ➡ Croissant ➡ +∞
f(x) = x² +∞ ↘ 0 ↗ +∞

↘ Décroissante sur ]−∞ ; 0]  |  ↗ Croissante sur [0 ; +∞[

⚠️ Pièges classiques à l'éval

√(x²) ≠ x — La bonne réponse : √(x²) = |x| (valeur absolue !)

Comparer sur les négatifs : si a < b < 0 alors a² > b² (l'ordre s'inverse !)
Exemple : −5 < −2 mais (−5)² = 25 > (−2)² = 4 ✅

x² = −4 → Pas de solution (x² est toujours ≥ 0)

✅ Résoudre x² = k
1
Si k < 0 → Aucune solution (x² ≥ 0 toujours)
2
Si k = 0 → x = 0 (unique solution)
3
Si k > 0 → x = √k ou x = −√k (deux solutions)
+
x² < k (k>0) → −√k < x < √k
+
x² > k (k>0) → x < −√k ou x > √k

🧠 Questions flash — tape pour révéler

Quelle est l'image de −3 par f(x) = x² ?
f(−3) = (−3)² = 9
Un négatif au carré devient positif !
👆 Clique pour voir la réponse
f(x) = x² est-elle croissante ou décroissante sur [−5 ; 0] ?
Décroissante sur ]−∞ ; 0], donc sur [−5 ; 0] aussi.
👆 Clique pour voir la réponse
Comparer 7² et 9² sans calculer (en utilisant les variations)
7 < 9 et les deux sont ≥ 0, x² est croissante sur [0;+∞[
→ 7² < 9² ✅ (49 < 81)
👆 Clique pour voir la réponse
Résoudre x² = 25
25 > 0 → x = 5 ou x = −5
👆 Clique pour voir la réponse

Fonction Racine Carrée

L'inverse du carré — uniquement sur les positifs

f(x) = √x
🌱 L'image pour comprendre

√x répond à la question : "Quel nombre, mis au carré, donne x ?"
√9 = 3 car 3² = 9. On ne peut pas prendre la racine d'un nombre négatif : impossible d'avoir a² = −4 !
La courbe commence à l'origine et monte de plus en plus doucement (comme une montagne qui s'aplatit).

📈 La courbe de √x — clique pour explorer !

👆 Clique sur la courbe pour voir l'image d'un x

📐 Domaine de définition

[0 ; +∞[
x doit être ≥ 0
Jamais négatif !

🪞 Parité

Ni paire ni impaire
Définie que sur [0;+∞[

↗ Variations

Croissante sur [0;+∞[
a < b → √a < √b

🎯 Propriété clé

(√x)² = x
Pour x ≥ 0 seulement

📊 Tableau de variations

x 0 +∞
f(x) = √x 0 ↗ ↗ ↗ +∞

↗ Croissante sur [0 ; +∞[  |  Minimum : 0 en x = 0

⚠️ Pièges classiques

√(x²) ≠ x mais √(x²) = |x| ← même piège que pour x² !

√9 = 3 (pas −3) — la racine est toujours positive ou nulle

√(a+b) ≠ √a + √b — erreur très fréquente !
Ex : √(9+16) = √25 = 5 ≠ √9 + √16 = 3+4 = 7

✅ Résoudre √x = k
1
Si k < 0 → Pas de solution (√x ≥ 0 toujours)
2
Si k ≥ 0 → x = k² (on met au carré les deux membres)
+
√x < k (k>0) → 0 ≤ x < k²
+
√x > k (k≥0) → x > k²

🧠 Questions flash

Calculer √36 et √(0,25)
√36 = 6 (car 6² = 36)
√(0,25) = 0,5 (car 0,5² = 0,25)
👆 Clique pour voir la réponse
Comparer √7 et √11 sans calculer
7 < 11 et √x est croissante → √7 < √11 ✅
👆 Clique pour voir la réponse
Résoudre √x = 4
k = 4 ≥ 0 → x = 4² = 16
👆 Clique pour voir la réponse
Simplifier √(49x²) pour x ≥ 0
√(49x²) = √49 × √(x²) = 7 × |x| = 7x (car x ≥ 0)
👆 Clique pour voir la réponse

Fonction Inverse

1 divisé par x — tout sauf zéro

f(x) = 1/x
🍕 L'image pour comprendre

Tu as 1 pizza à partager entre x personnes. Plus il y a de personnes (x grand), moins chacune reçoit (1/x petit). Et on ne peut jamais diviser par 0 — c'est impossible mathématiquement. La courbe a deux branches qui ne se touchent jamais les axes.

📈 La courbe de 1/x — l'hyperbole !

👆 Clique pour voir les valeurs

📐 Domaine de définition

ℝ* = ]−∞;0[ ∪ ]0;+∞[
Tout sauf 0 !

🪞 Parité

IMPAIRE
f(−x) = −f(x)
Symétrie / O

↘ Variations

Décroissante sur chaque intervalle
]−∞;0[ ET ]0;+∞[

🎯 Asymptotes

x = 0 et y = 0
Les axes sont des "murs"

📊 Tableau de variations

x −∞ 0 +∞
f(x) = 1/x 0⁻ ↘ ∥ ↘ 0⁺

↘ Décroissante sur ]−∞;0[  |  ↘ Décroissante sur ]0;+∞[

⚠️ LE PIÈGE MAJEUR

1/x est décroissante sur chaque intervalle séparément, mais on ne peut pas dire qu'elle est décroissante sur ℝ* entier !

Pourquoi ? Car −1 < 1 mais f(−1) = −1 et f(1) = 1, donc f(−1) < f(1). Si elle était vraiment décroissante sur ℝ*, on aurait f(−1) > f(1). Contradiction !

L'ordre s'inverse : 0 < a < b → 1/a > 1/b (plus x est grand, 1/x est petit)

✅ Résoudre 1/x = k (k ≠ 0)
1
x = 1/k (une seule solution)
⚠️
Pour 1/x > k : mettre tout d'un côté → 1/x − k > 0 → réduire au même dénominateur → étude de signe !

🧠 Questions flash

Calculer f(0,5) et f(−4) pour f(x) = 1/x
f(0,5) = 1/0,5 = 2
f(−4) = 1/(−4) = −0,25
👆 Clique pour voir la réponse
Comparer 1/3 et 1/7 (sans calculer)
0 < 3 < 7 et 1/x est décroissante sur ]0;+∞[
→ 1/3 > 1/7 ✅
👆 Clique pour voir la réponse
1/x est-elle croissante ou décroissante sur ℝ* ?
Ni l'un ni l'autre ! Elle est décroissante sur chaque intervalle ]−∞;0[ et ]0;+∞[ séparément, mais pas sur ℝ* entier.
👆 Clique pour voir la réponse
f(x) = 1/x est-elle paire ou impaire ?
Impaire : f(−x) = 1/(−x) = −1/x = −f(x) ✅
La courbe est symétrique par rapport à l'origine O.
👆 Clique pour voir la réponse

Fonction Cube

Multiplie un nombre par lui-même trois fois

f(x) = x³
📦 L'image pour comprendre

Imagine un cube de côté x. Son volume vaut x × x × x = x³. Contrairement au carré, x³ peut être négatif (un cube de côté −2 donne −8). La courbe en S traverse tout ℝ de −∞ à +∞.

📈 La courbe de x³ — la belle courbe en S !

👆 Clique pour voir les valeurs

📐 Domaine de définition

ℝ (tous les réels)
x peut être n'importe quoi

🪞 Parité

IMPAIRE
f(−x) = −f(x)
Symétrie / O

↗ Variations

Croissante sur ℝ
a < b → a³ < b³ toujours

🎯 Clé unique

x³ = k → x = ∛k
Toujours 1 seule solution

📊 Tableau de variations

x −∞ +∞
f(x) = x³ −∞ ↗ ↗ ↗ +∞

↗ Strictement croissante sur ℝ entier — l'ordre est toujours conservé !

✅ Résoudre x³ = k
1
Toujours exactement une seule solution : x = ∛k
2
Exemples : x³ = 8 → x = 2  |  x³ = −27 → x = −3
+
Comparer : a < b → a³ < b³ (l'ordre est toujours conservé, peu importe les signes)

🧠 Questions flash

Calculer f(−2) et f(3) pour f(x) = x³
f(−2) = (−2)³ = −8
f(3) = 3³ = 27
👆 Clique pour voir la réponse
Comparer (−5)³ et (−3)³
−5 < −3 et x³ est croissante sur ℝ
→ (−5)³ < (−3)³ (soit −125 < −27) ✅
👆 Clique pour voir la réponse
Résoudre x³ = −64
x = ∛(−64) = −4 (car (−4)³ = −64) ✅
👆 Clique pour voir la réponse
Différence clé entre x² et x³ sur les négatifs ?
x² est toujours positif (même pour x négatif)
x³ garde le signe de x : si x < 0 alors x³ < 0 !
👆 Clique pour voir la réponse

📋 Récap Total

Tout en un seul coup d'œil

🟥 f(x) = x²
  • Domaine : ℝ
  • Parité : PAIRE (f(−x) = f(x))
  • ↘ sur ]−∞;0] puis ↗ sur [0;+∞[
  • Min = 0 en x = 0
  • x² ≥ 0 toujours !
  • √(x²) = |x| (pas x !)
🟩 f(x) = √x
  • Domaine : [0 ; +∞[
  • Ni paire ni impaire
  • ↗ sur [0 ; +∞[
  • √x ≥ 0 toujours
  • (√x)² = x (x≥0)
  • √(a+b) ≠ √a+√b !
🟣 f(x) = 1/x
  • Domaine : ℝ* (tout sauf 0)
  • Parité : IMPAIRE
  • ↘ sur ]−∞;0[ et ↘ sur ]0;+∞[
  • Pas définie en 0 !
  • Asymptotes : x=0 et y=0
  • L'ordre s'inverse sur les >0
🟡 f(x) = x³
  • Domaine : ℝ
  • Parité : IMPAIRE
  • ↗ sur ℝ entier
  • x³ peut être négatif
  • 1 seule solution : x=∛k
  • L'ordre est toujours conservé

🔑 Méthode comparer f(a) et f(b) — toujours la même logique

Étape Action
1 Identifier sur quel intervalle sont a et b
2 Regarder si f est ↗ ou ↘ sur cet intervalle
3 ↗ : a < b → f(a) < f(b)  |  ↘ : a < b → f(a) > f(b)
